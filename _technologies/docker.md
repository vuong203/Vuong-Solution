# Docker

## Description détaillée
Docker est une plateforme de conteneurisation open-source qui permet de créer, déployer et exécuter des applications dans des conteneurs légers.

## Avantages
- Isolation des environnements
- Portabilité des applications
- Facilité de mise à l'échelle

## Inconvénients
- Courbe d'apprentissage pour les débutants
- Gestion des images volumineuses

## Cas d'usage réels
Utilisé pour déployer des applications web en développement et en production.
