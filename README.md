# Site Technologique Personnel

## Mise à jour hebdomadaire

- **Semaine 1** : Ajouter un nouveau cours sur [Sujet]
- **Semaine 2** : Évaluer une nouvelle technologie
- **Semaine 3** : Critiquer une application
- **Semaine 4** : Mettre à jour contenu existant

## Déploiement
1. Créez un repo GitHub nommé 'site-tech'
2. Poussez le code
3. Activez GitHub Pages dans les settings du repo
4. Le site sera disponible à https://username.github.io/site-tech

## Mise à jour sans codage
Utilisez un éditeur Markdown en ligne ou GitHub pour éditer les fichiers .md directement.
