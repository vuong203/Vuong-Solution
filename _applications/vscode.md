# Visual Studio Code

## Critique
VS Code est un éditeur de code gratuit et open-source développé par Microsoft, excellent pour le développement.

Comparaison avec Sublime Text : VS Code offre plus d'extensions intégrées.

## Tutoriel d'utilisation étape par étape
1. Téléchargez et installez VS Code depuis le site officiel.
2. Ouvrez un projet.
3. Installez des extensions via le marketplace.

## Avantages et inconvénients
- Avantages : Léger, extensible
- Inconvénients : Consommation de ressources parfois élevée
