# GitHub Desktop

## Critique
GitHub Desktop simplifie l'utilisation de Git et GitHub avec une interface graphique.

Comparaison avec GitKraken : GitHub Desktop est plus intégré à GitHub.

## Tutoriel d'utilisation étape par étape
1. Téléchargez et installez GitHub Desktop.
2. Connectez votre compte GitHub.
3. Clonez un repo.

## Avantages et inconvénients
- Avantages : Facilité d'utilisation
- Inconvénients : Limité pour tâches avancées
